# 로또 번호 추첨 사이트

## 개요

이 애플리케이션은 사용자가 로또 번호를 생성할 수 있는 간단한 웹사이트입니다. 버튼을 클릭하면 1부터 45까지의 숫자 중 6개의 고유한 숫자를 무작위로 추첨하여 화면에 표시합니다.

## 기능

-   **번호 생성:** "번호 생성" 버튼을 클릭하면 새로운 로또 번호 6개를 생성합니다.
-   **번호 표시:** 생성된 번호는 시각적으로 구분되는 스타일로 화면에 표시됩니다.

## 디자인

-   **레이아웃:** 중앙 정렬된 간단한 레이아웃을 사용합니다.
-   **색상:** 현대적이고 시각적으로 매력적인 색상 팔레트를 사용합니다.
-   **글꼴:** 가독성이 좋은 글꼴을 사용합니다.
-   **애니메이션:** 번호가 생성될 때 부드러운 애니메이션 효과를 추가하여 사용자 경험을 향상시킵니다.

## 기술 스택

-   HTML
-   CSS
-   JavaScript
